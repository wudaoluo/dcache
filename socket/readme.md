### 协议设计

> request

|0000 0000|0000 0000|0000 0000|0000 0000|0000 0000|0000 0000|--------|0000 0000|0000 0000|------------|
|--------length-----|-req-op---|   保留  |-------------------|key data|------value len----| value data |  
                                           key长度
                      
        
> response
|0000 0000|0000 0000|0000 0000|0000 0000|0000 0000|0000 0000|--------|0000 0000|0000 0000|------------|
|--------length-----|-resp-op-|   保留   |----keylen    -----|key data|------value len----| value data |  
                                                                或者 err 长度        err          
                      
### req op 和 http 保留一致
    0 保留
    1 Get 
    2 PUT
    3 Del
    
    
### resp op 
    0 保留
    1 成功
    2 没有找到
    3 服务器返回指定ip
